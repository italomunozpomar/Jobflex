Módulo Página Principal Candidato
ID
Requerimientos
RF-17
El sistema debe mostrar un panel de bienvenida que incluya el nombre del candidato, su avatar, el total de ofertas vistas y el total de postulaciones realizadas.
RF-18
El sistema debe mostrar botones para “Crear CV” y “Subir CV”, permitiendo al candidato acceder directamente a la vista correspondiente para crear un nuevo CV o subir un archivo PDF/DOC de CV.
RF-19
El sistema debe permitir al usuario buscar ofertas laborales desde la página principal mediante una barra de búsqueda, filtros y enlaces a búsquedas populares.
RF-20
El sistema debe mostrar un carrusel de ofertas destacadas para el candidato, permitiendo acceder al detalle de cada oferta mediante el botón “Ver oferta”.
RF-21
El sistema debe mostrar las postulaciones más recientes del candidato, incluyendo cargo, fecha de postulación y estado, además de permitir acceder a la vista completa mediante el botón “Ver todas”.
RF-22
El sistema debe permitir al candidato acceder desde la página principal a las vistas de Mi Perfil, Mis Postulaciones, Mis Perfiles Profesionales, Configuración y Cerrar Sesión.
